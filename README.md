# Sahil - Ethical Hacker Portfolio

An advanced hacker-themed portfolio website showcasing ethical hacking skills and expertise.

## Features

- **Terminal Interface**: Interactive command-line style interface
- **Matrix Effects**: Animated matrix rain and binary code backgrounds
- **Glitch Effects**: Random glitch animations for authentic hacker aesthetic
- **Responsive Design**: Fully responsive across all devices
- **Interactive Terminal**: Functional terminal with multiple commands
- **Social Links**: Integrated social media connections
- **Skills Showcase**: Animated skill bars for security expertise

## Setup Instructions

1. **Add Your Profile Picture**:
   - Replace `profile.jpg` with your actual profile picture
   - Place it in the same directory as `sahil.html`

2. **Update Social Media Links**:
   - Open `sahil.html`
   - Replace the placeholder URLs with your actual social media profiles:
     - Line 51: `https://facebook.com/yourprofile`
     - Line 56: `https://instagram.com/yourprofile`
     - Line 61: `https://linkedin.com/in/yourprofile`
     - Line 66: `https://github.com/yourprofile`

3. **Customize Information**:
   - Update your name in line 39 if needed
   - Modify skill percentages in the `data-level` attributes (lines 87, 103, 119, 135, 151)
   - Adjust skill descriptions in the `<ul>` sections

## Terminal Commands

The portfolio includes an interactive terminal. Try these commands:
- `help` - Show available commands
- `about` - Display about information
- `skills` - Show core competencies
- `contact` - Display contact information
- `clear` - Clear terminal output
- `whoami` - Display user identity
- `date` - Show current date/time
- `matrix` - Matrix animation effect
- `ls` - List directory contents
- `pwd` - Show current directory
- `ping google.com` - Simulated ping command
- `nmap -sV localhost` - Simulated nmap scan

## Security Note

This portfolio is designed for demonstration purposes and does not contain any actual security vulnerabilities. All terminal commands are simulated and do not perform real system operations.

## Customization Tips

1. **Color Scheme**: Modify CSS variables in `:root` section of `style.css`
2. **Animations**: Adjust timing in keyframe animations
3. **Commands**: Add new commands to the `commands` object in `script.js`
4. **Skills**: Update skill categories and percentages in HTML
5. **Typography**: Change fonts via Google Fonts link in HTML

## Deployment

Simply upload all files to your web server or hosting platform. The portfolio works entirely client-side with no server requirements.
